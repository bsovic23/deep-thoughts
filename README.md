# deep-thoughts

## Project Description

This project is a dashboard that allows users to create a profile/login and post thoughts.

Users can add other users as friends, and react to other individual thought posts

## Tools Used

* Node.js
* Express.js
* React.js
* MongoDB
* npm:
    * apollo
    * graphql
    * concurrently
